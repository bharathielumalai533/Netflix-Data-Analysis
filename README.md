# Netflix-Data-Analysis
A comprehensive data analysis project on the Netflix Movies and TV Shows dataset using Python, Pandas, NumPy, Matplotlib, and Seaborn. The project includes data cleaning, transformation, exploratory data analysis (EDA), visualizations, and business insights.
